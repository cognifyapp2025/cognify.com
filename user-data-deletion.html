Cognify • User Data Deletion Instructions

Huling update: 15 September 2025

Layunin

Ang pahinang ito ay tumutupad sa Facebook “User Data Deletion” requirement at nagsisilbing gabay ng lahat ng Cognify users para mag-request ng pagbura ng data.

Opsyon A: In-app “Delete Account” (kung available)

  1. Buksan ang Cognify app.
  2. Pumunta sa Settings > Account.
  3. Piliin ang Delete Account at sundin ang prompt.

Resulta: Ide-delete namin ang account at nauugnay na data sa production database.

Opsyon B: Facebook Login data deletion request

  1. Facebook app/website: Settings & Privacy > Settings > Apps and Websites > Logged in with Facebook.
  2. Piliin ang Cognify at i-Remove.
  3. Mag-email ng request (Opsyon C) para burahin ang data sa aming database.

Opsyon C: Email request

I-email ang opisyal na contact namin. Ilakip:

  • Pangalan sa account at login email
  • Auth provider (Facebook, Google, o Email/Password)
  • Kung Facebook user: optional ang iyong Facebook User ID kung alam mo

Template:

Subject: Cognify Data Deletion Request
Body:
- Full name:
- App login email:
- Auth provider:
- Karagdagang detalye (optional):

Target completion: sa loob ng 30 araw mula pagtanggap ng kumpletong request.

Saklaw ng pagbura

  • Account identifiers at auth links
  • User profile fields (hal. pangalan, edad kung ibinigay)
  • App data: progreso at stats ng laro

Hindi binubura

  • Data na kailangan ayon sa batas kung meron
  • Aggregated o anonymized analytics na hindi maiuugnay sa iyo

Official contact

Ipadala ang requests sa cognifyapp2025@gmail.com.

Tandaan
  • Ang pag-remove ng Cognify sa Facebook Apps and Websites ay hindi awtomatikong nagbubura ng data sa aming database. Gumamit ng Opsyon A o C.
  • Wala kaming automated “data deletion callback” endpoint sa ngayon.
